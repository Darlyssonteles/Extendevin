# Privacy Policy for extenDevin

**Last updated: November 10, 2024**

## Overview
extenDevin is a Chrome extension that integrates Devin AI with Azure DevOps to provide AI-powered development assistance.

## Data Collection and Usage

### Data We Access
- **Azure DevOps Content:** Work items, pull requests, code diffs, and comments from dev.azure.com
- **User Input:** Chat messages and queries entered by users
- **Local Preferences:** Theme settings and configuration stored in browser storage

### How We Use Data
- Work item and PR data is transmitted to Devin AI API to provide context-aware assistance
- Chat messages are sent to Devin AI API to generate responses
- All data transmission occurs only when users explicitly interact with the extension
- Preferences are stored locally in your browser only

### Data Transmission
- All data is transmitted securely via HTTPS to api.devin.ai
- We do NOT store any user data on our own servers
- Conversation history is stored locally in your browser storage only
- No data is shared with third parties except Devin AI API

### Third-Party Services
- **Devin AI API:** Processes user queries and Azure DevOps content. See Devin's privacy policy at https://devin.ai/privacy

## Data Storage
- All conversation history is stored locally in your browser using Chrome's storage API
- You can clear this data at any time through the extension settings
- Uninstalling the extension removes all locally stored data

## Permissions Explanation
- **dev.azure.com access:** Read work items and PRs to provide context
- **devin.ai access:** Communicate with Devin AI services
- **storage:** Save user preferences and conversation history locally
- **tabs:** Detect Azure DevOps pages to activate the extension

## User Control
- Extension only activates on Azure DevOps pages
- AI assistant only accesses page content when you open the chat
- You can disable or uninstall the extension at any time
- All local data can be cleared through extension settings

## Security
- All communications use HTTPS encryption
- No passwords or authentication tokens are stored
- Extension uses your existing Azure DevOps and Devin authentication
